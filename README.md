# Sistema de Gestión de Trabajadores - MYPER Software

## 🚀 Descripción
Sistema web desarrollado para la prueba técnica de Analista Programador .NET en MYPER Software.

## 🛠️ Tecnologías
- ASP.NET Core 8.0
- Entity Framework Core
- SQL Server
- Bootstrap 5.3
- JavaScript ES6+

## 📋 Funcionalidades
- ✅ CRUD completo de trabajadores
- ✅ Diseño responsive profesional
- ✅ Filtros y búsqueda en tiempo real
- ✅ Validaciones avanzadas
- ✅ Procedimientos almacenados

## 🏗️ Arquitectura
- Patrón MVC
- Repository Pattern
- Dependency Injection
- Code First Migrations

## 🚀 Instalación
1. Ejecutar `ScriptBD.sql`
2. Configurar connection string en `appsettings.json`
3. Ejecutar `Update-Database`
4. Run application
